Purpose

Use the Accessible University site to:

  1. Demonstrate common web accessibility principles at trainings, presentations, and workshops on accessible web design.
  2. Learn common web accessibility problems and solutions in an easy-to-understand way.

Updates from the original Accessible University

This example website was inspired by the original Accessible University. This new version includes updates to the look, feel, and code. It has been adapted to use the Twitter Bootstrap framework. Many components in the Bootstrap framework provide an accessible base to work from.
